# PHP Info Plugin

Ein b1gMail Plugin, das phpinfo() in schön formatierter Form im Admin-Bereich anzeigt.

## Beschreibung

Das PHP Info Plugin ermöglicht es Administratoren, PHP-Konfigurationsinformationen direkt im b1gMail Admin-Bereich anzuzeigen. Die Ausgabe wird übersichtlich und benutzerfreundlich formatiert.

## Funktionen

- **Schön formatierte PHP Info**: Zeigt phpinfo() in einem übersichtlichen, modernen Design
- **Admin-Integration**: Vollständig in den b1gMail Admin-Bereich integriert
- **Responsive Design**: Optimiert für verschiedene Bildschirmgrößen
- **Scrollbare Ausgabe**: Bei großen phpinfo-Inhalten mit Scrollbar
- **Schnelle Navigation**: Einfacher Zugriff über den Admin-Bereich

## Installation

1. Kopieren Sie den gesamten `PHP Info` Ordner in das `plugins` Verzeichnis Ihrer b1gMail Installation
2. Gehen Sie zum Admin-Bereich von b1gMail
3. Navigieren Sie zu "Plugins" und aktivieren Sie das "PHP Info" Plugin
4. Das Plugin ist nun im Admin-Bereich unter "PHP Info" verfügbar
